# 대구 버스 0.7.2

## 개선 사항

- 즐겨찾기 추가·해제, 노선 고정, 순서 저장, 도착정보 수동 갱신 결과를 화면 이동과 관계없이 확인할 수 있다.
- 즐겨찾기를 해제한 직후 `실행 취소`를 누르면 정류장, 고정 노선, 해당 위젯 연결을 함께 복원한다.
- 검색 결과와 정류장 상세 화면의 즐겨찾기 버튼이 실제 저장 상태와 처리 중 상태를 표시하며, 중복 입력을 막는다.
- 검색창에 검색어 지우기 버튼과 키보드 검색 완료 동작을 추가했다.
- 정류장·노선·장소·주변 검색 실패와 API 키 저장, 업데이트 확인 및 다운로드 결과를 명확하게 안내한다.
- 자동 갱신은 조용히 유지하고 사용자가 누른 수동 갱신만 진행 및 성공·실패 상태를 표시한다.
- 위젯 정류장 연결 중에는 선택을 잠그고 진행 상태를 표시하며, 실패하면 화면에서 바로 다시 시도할 수 있다.
- 위젯 새로고침을 누르는 즉시 `업데이트 중` 상태로 전환하고 이전 실패 문구를 정리한다.
- 즐겨찾기 순서 변경 화면에 순서 번호와 이동 가능 상태를 표시하고, 더 이동할 수 없는 버튼은 비활성화한다.
- 메뉴의 정보 항목을 비대화형으로 정리하고 캐시 갱신·업데이트 확인 중 중복 실행을 막는다.

## 기존 데이터와 호환성

0.7.0 또는 0.7.1에서 저장한 공공데이터 API 키, 즐겨찾기 정류장, 고정 노선, 도착 캐시와 위젯 연결을 그대로 유지한다. 데이터베이스 스키마 변경은 없다.

## 운영 설정

- 기존과 동일하게 Cloudflare Worker의 `NAVER_SEARCH_CLIENT_ID`, `NAVER_SEARCH_CLIENT_SECRET`과 GitHub Actions의 `PLACE_SEARCH_BASE_URL` repository variable이 필요하다.
- Naver client secret과 Android 지도 Key ID는 APK 또는 Git 저장소에 직접 넣지 않는다.
