# 대구 버스 0.7.0

## 새 기능

- 출근·퇴근 슬롯을 최대 20개의 즐겨찾기 정류장으로 교체했다.
- 버스·정류장·장소 통합 검색과 현재 위치/장소 주변 정류장 지도를 추가했다.
- 정류장 전체 노선 도착정보와 접근 중인 모든 초정밀 차량을 한 지도에서 볼 수 있다.
- 정류장 상세에서 자주 타는 노선을 고정하면 홈과 위젯이 해당 노선을 우선 표시한다.
- 정류장 상세 지도는 첫 프레임부터 선택 정류장을 표시한다.
- 위젯을 즐겨찾기 정류장에 연결하고 크기에 따라 2개 또는 4개 노선을 표시한다.
- 햄버거 메뉴에 즐겨찾기 순서, API 키, 위치 상태, 캐시, 호출량, 공지와 업데이트 진입점을 추가했다.

## 데이터 전환

업데이트 시 기존 출근·퇴근 즐겨찾기, 슬롯 도착 캐시와 이전 위젯 연결은 삭제된다. 공공데이터 API 키와 노선·경로·차량 캐시는 유지된다.

## 운영 준비 필요

장소 검색을 활성화하려면 Cloudflare Worker에 Naver Local Search client ID/secret을 secret으로 저장해 배포하고, 앱 빌드의 `PLACE_SEARCH_BASE_URL`에 Worker URL을 주입해야 한다.
